# GDSC-Workshop-1

#### Clone the project:

```bash
git clone https://github.com/M4r0uan3/GDSC-Workshop-1.git
```

#### Cd into the folder:

```bash
cd GDSC-Workshop-1
```

#### Change the file permisions:

```bash
cd kali-setup
chmod +x setup.sh
```

#### Run the script:

```bash
sudo ./setup.sh
```
